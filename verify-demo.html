No code demo of linking to Verify

If you have images that have Content Credentials, for example ones you saved using Photoshop with Content Credentials (Beta) enabled or created with Adobe Firefly, this is the simplest possible way to link them to Verify to display the Content Credentials.

Simply add this code in the head of your page:

    window.addEventListener('load', function() {
      var images = document.getElementsByTagName('img');
      for (var i = 0; i < images.length; i++) {
        images[i].onclick = function() {
          window.open('https://contentcredentials.org/verify?source=' + this.src, '_blank');
        };
      }
    });

If you want to indicate whether an image has Content Credentials, you could, for example, simply add a credentials attribute to the img tags for the image, then check for the existence of that attribute: 

  window.addEventListener('load', function() {
    var images = document.getElementsByTagName('img');
    for (var i = 0; i < images.length; i++) {
      if (images[i].hasAttribute('credentials')) {
        images[i].onclick = function() {
          window.open('https://contentcredentials.org/verify?source=' + this.src, '_blank');
        };
      } else {
        images[i].onclick = function() {
          alert('No Content Credentials indicated for this image.');
        };
      }
    }
  });
